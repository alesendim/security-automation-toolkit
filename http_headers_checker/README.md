# HTTP Security Headers Checker

Esta ferramenta tem como objetivo verificar a presença de cabeçalhos de
segurança HTTP em aplicações web, auxiliando na identificação de más
configurações comuns.

## Objetivo
Demonstrar conhecimentos básicos sobre segurança em aplicações web,
funcionamento do protocolo HTTP e automação de verificações simples.

## Cabeçalhos de segurança analisados
- Content-Security-Policy
- X-Frame-Options
- X-Content-Type-Options
- Strict-Transport-Security
- Referrer-Policy

## Status
Em desenvolvimento.

## Uso futuro
Esta ferramenta poderá ser utilizada para:
- Auditorias básicas de segurança web
- Validação automática de configurações
- Apoio a atividades de SOC e Blue Team
