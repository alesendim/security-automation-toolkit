# Password Strength Checker

Esta ferramenta tem como finalidade avaliar a força de senhas com base em
critérios comuns de segurança da informação.

## Objetivo
Demonstrar boas práticas relacionadas à criação e validação de senhas seguras.

## Critérios considerados
- Tamanho mínimo
- Uso de letras maiúsculas e minúsculas
- Números
- Caracteres especiais

## Status
Em desenvolvimento.

## Uso futuro
Pode ser integrada a sistemas de cadastro, validação de usuários ou auditorias
de segurança.
