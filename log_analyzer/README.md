# Log Analyzer

Esta ferramenta é destinada à análise básica de arquivos de log, com foco na
identificação de padrões suspeitos ou eventos relevantes para segurança.

## Objetivo
Demonstrar compreensão sobre análise de logs, uma atividade essencial em
operações de SOC e monitoramento de segurança.

## Exemplos de uso
- Identificação de tentativas de login suspeitas
- Busca por mensagens de erro recorrentes
- Detecção de padrões simples de ataque

## Status
Em desenvolvimento.

## Uso futuro
Pode ser adaptada para analisar logs de servidores web, sistemas Linux ou
ferramentas de segurança.
