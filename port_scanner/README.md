# Port Scanner

Esta ferramenta realiza um escaneamento básico e não intrusivo de portas TCP
em um host específico, com o objetivo de identificar serviços potencialmente
expostos.

## Objetivo
Demonstrar conhecimentos fundamentais de redes, comunicação TCP/IP e automação
de tarefas comuns em ambientes de SOC e cibersegurança defensiva.

## Funcionamento
O script tenta estabelecer conexão com um conjunto de portas comuns e informa
quais estão abertas.

## Como executar
```bash
python port_scanner.py
