# Security Automation Toolkit

Este projeto contém uma coleção de scripts em Python focados na automação
de tarefas comuns de cibersegurança e operações de SOC.

O objetivo é demonstrar conhecimentos práticos em segurança da informação,
habilidades de programação em Python e capacidade de resolução de problemas
para vagas de nível inicial e estágio.

## Ferramentas incluídas
- Scanner de portas (básico e não intrusivo)
- Verificador de cabeçalhos de segurança HTTP
- Analisador de logs
- Verificador de força de senhas

## Tecnologias
- Python 3
- Fundamentos de redes
- Boas práticas de segurança

## Aviso
Todas as ferramentas são destinadas exclusivamente a fins educacionais
e de segurança defensiva.
